Types of Developers

WEB

  • The role of web developer is to design code and modify website from layout to functions and according to client’s specifications
  • They strive to create visually appealing sites that features user friendly design and clear navigation
  • They must have a strong understanding of UI, cross -browser compatibility, general web functions and standards.
  • Create and maintain software documentation
  • Be responsible for maintaining, expanding, and scaling the sites
  • Must handle both server-side and front-end logic
  • They usually involves implementing all the visual elements that users see and use in the web applications
  • They work close with the project manager and designers to ensure the final product adheres to the predetermined budget, scope and design.
  • Web developers are found working in various types of organization including large corporations and government small and medium-sized companies,or alone as freelancers

GAME

  • Game designers are an important part of comprehensive team of designers and developers that coordinate the complex task of creating a new video game.
  • Game designers devise what a game consists of and how it plays.
  • They plan and define all the elements of a game
  • The game designer comes up with the games premises.
  • Most of the core ingredient are already defined and they must decide how to create the best game using these elements, within a certain budget and timescale.
  • Game designers are employed by development studios, both independent and publisher-owned.
  • The game design process is usually shared between a number of different people, overseen by a lead designer
  • Game Designers should have a deep understanding of the capabilities and benefits of different hardware platforms as well as similarity with software technologies and techniques appropriate to each platform.
